Help

The search bar at the bottom of each page has two modes of operation:

  • Keyword search
  • Semantic search

For keyword search you simply type the keywords you are seeking and click go or press enter, e.g. Controlled English

By default any phrase that you type is searched for exactly, but you can specify various guidance to the search engine:

  • Quoted values to identify a specific phrase, e.g. "Controlled English"
  • NOT operator: To exclude the phrase that follows the operator, e.g. "Controlled English" NOT Gaian
  • OR operator: To optionally include the phrase that follows the operator, e.g. "Controlled English" OR CNL
  • AND operator: To required the phrase that follows the operator, e.g. "Controlled English" AND "coalition operation"

These operators can be used in any configuration and amount, e.g. "Controlled English" AND Gaian NOT network

Both keyword and semantic search are not case sensitive

The current semantic search capability is limited but enables you to specify the name of an instance such as David Mott and optionally a characteristic related to that instance, such as co authors.

For example "Show me David Mott co-authors".

Both keyword search and semantic search will navigate directly to the correct science library page if onl a single match is found, otherwise a list of matches are shown.
